About Me

Software (Programa)

Software mak programa/aplikasaun ne'ebé lao iha komputadór ka telemovel.

  • Windows
  • Microsoft Word
  • Photoshop
  • WhatsApp

Software sei iha parte 3:

  • System Software → hanesan OS (Windows, Linux, Android)
  • Application Software → hanesan Word, Excel, Browser, Game
  • Utility Software → hanesan Antivirus, Cleaner

Hardware (Aparatu Físiku)

Hardware mak hanesan komponente físiku ne'ebé bele kear. Exemplu:

  • Monitor
  • Teclado (Keyboard)
  • Rato (Mouse)
  • CPU, Harddisk, Printer
